# scl - system control language

SCL is an attempt to reinvent a proper shell

## Goals
1. Simplicity
2. Consistency
3. Flexibility
4. Useability
5. Backwords compatibility

## Compile &Install
You need to have antlr, meson, ninja and a c++11 compiler (clang, gcc) installed on your system

```
meson builddir
cd builddir
ninja
```
